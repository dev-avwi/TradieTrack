Here's the comparison:

NEW DESIGN (Phase 2 Roadmap)
What it adds:

Feature	What It Does

Client Portal	Web page where client sees everything
OTP Recovery	Client can access with phone + code if link is lost
Two-Way Messaging	Client can message tradie from portal

4. Two-Way Communication

5. Automatic Lead Capture

7. Recovery When Links Are Lost
OTP is more secure than competitors (Jobber just uses last 4 of phone).

On Social Media (Facebook/Instagram):
